javascript (NO CONFUNDIR CON JAVA) es un lenguaje de programacion de alto nivel, ligero, interpretado y multiparadigma (Aunque se suele decir que esta orientado a objetos)

¿Que es alto nivel (o bajo?)?
Basicamente el lenguaje de mas bajo nivel que existe, es el binario, que se traduce en impulsos electricos
Cuanto más parecido es a nuestro lenguaje (El "humano") más de alto nivel es, y, cuanto más bajo nivel es, más abstracto es el lenguaje

Ejemplo de binario:

1001 <- 9 en binario
9 <- 9 en javascript

¿Que significa que sea interpretado?
Tenemos a grandes rasgos 2 tipos de lenguaje: Interpretado y Compilado.
El interpretado se va ejecutando y leyendo, esto conlleva mas periodos de carga, pero mas ligero cada uno.
El lenguaje compilado se lee todo, y luego se ejecuta (Mas tiempo de carga inicial, menos cargas posteriores). por ejemplo cuando juego al Hollow Knigth y al principio tarda en arrancar para cargar y luego arranca.

Eligiremos uno u otro dependiendo del uso que le vamos a dar a nuestro programa.

-Funcion principal de Javascript: Proporciona dinamismo e interactividad a los sitios web, valida formularios
y gestiona elementos interactivos, funcionando en el lado del cliente.

-Interactividad con el DOM: Interactua con el Document Object Model (DOM) para modificar la estructura de la pagina sin recargarla

-Uso versatil: Aunque nacio para navegadores, hoy en dia se utiliza tambien en el desarrollo backend (Con Node.js) aplicacioens moviles y juegos

-Popularidad: ES uno de los lenguajes más utilizados en la actualidad, su uso es fudnamental para crear páginas web.

¿Que es una variable?
Es un espacio donde guardamos información

Ejemplo (En JS):

var dato = 3

var -> Crea variable, le indica a JS que a continuación vamos a declarar una variable
dato -> Nombre con el que vamos a identificar nuestra variable
3 -> Valor que le asignamos a la variable
